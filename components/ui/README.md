# components/ui

Domain-agnostic UI primitives. These components have no business logic — they're reusable building blocks.

| Component | Purpose |
|-----------|---------|
| `Avatar.tsx` | User avatar (image or initials fallback) |
| `Card.tsx` | Generic card container (default/glass/interactive variants) |
| `SearchBar.tsx` | Search input with magnifying glass icon |
| `Logo.tsx` | LogoFull and LogoSmall |
| `ThemeToggle.tsx` | Dark/light theme toggle |
| `FeatureInProgress.tsx` | Overlay for features under development |
| `Navbar.tsx` | Top navigation bar |
| `Sidebar.tsx` | Side navigation (Dashboard, Projects, API) |
| `modal/Modal.tsx` | Generic centered modal with accept/cancel |
| `modal/RightModal.tsx` | Slide-in right panel modal |
| `toast/ToastProvider.tsx` | Toast context + provider |
| `toast/useToast.ts` | Hook to access toast context |
| `stepper/Stepper.tsx` | Multi-step form wizard |
