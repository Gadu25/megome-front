# types

Domain type definitions organized by concern:

| File | Contains |
|------|----------|
| `domain.ts` | Entity types — Profile, Skill, Experience, Education, Project, Certificate, Technology, PersonalAccessToken, ApiUsageLog |
| `form.ts` | Form/input payloads — ProfileForm, SkillForm, ExperienceForm, EducationForm, CertificateForm, ProjectForm, etc. |
| `api.ts` | API response/request types — LoginResponse, ApiUsage, DashboardOverview, etc. |
| `ui.ts` | UI-only state types — ProjectImage, Image |
