# features

Domain-scoped feature folders — each contains its own components, schemas, and a barrel `index.ts`:

| Feature | Contents |
|---------|----------|
| `auth/` | Login, register, logout, Google OAuth |
| `profile/` | Profile display, skills, education, experience, certificates — including CRUD forms |
| `project/` | Project CRUD wizard, project listing, detail view |
| `settings/` | Account settings, API tokens, integrations, security |
| `api/` | API documentation pages (intro, reference) |
