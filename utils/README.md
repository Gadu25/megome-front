# utils

Pure utility functions grouped by domain:

| Directory | Purpose |
|-----------|---------|
| `date/` | Date formatting and calculations (formatDate, humanizeDate, calculateAge) |
| `api/` | API request helpers (handleResponse, withRequest) |
| `ui/` | UI display helpers (getStatusStyle, mapProjectImagesToUI) |
| `env/` | Runtime environment checks (isClientSide) |
